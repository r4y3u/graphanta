# Graphanta v0.2.0-alpha.2

数学的な思考と表現を支援する、ローカルファーストの作図アプリです。

## 起動

ZIPを展開し、`index.html`または`Graphanta.html`をEdge／Chromeで開きます。開発環境では`npm ci`後に`npm run dev`、公開用生成は`npm run build`です。

## 今回の主な追加

- 画面比率と倍率に追従し、プロット領域全体へ続く方眼・座標軸
- 正方向の軸端に`x`、`y`を表示
- 右クリックで現在のツールグループをポインター位置へ表示
- 右ドラッグによる一時スクロール
- 2本指タップ／移動／ピンチを排他的に判定するタッチ操作
- アレー図グループへ「玉」「人」を追加
- 玉・人の連続クリック配置、ドラッグによるサイズ指定と次回への継承
- インスタント／4ショット／クリップボードのスクリーンショットメニュー
- 選択図形を直接回転するキャンバス上の回転ハンドル

詳細は`HANDOFF.md`、`DEVELOPMENT_PLAN.md`、`TEST_CHECKLIST.md`を参照してください。
