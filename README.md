# Graphanta v1 alpha

数学的な思考・表現を支える、ローカルファーストの図・グラフ作成Webアプリです。

## 現在の段階

`0.1.0-alpha.1` は、v1.0.0へ向けた最初の動作版です。ラフで示された構成をもとに、プロットエリア、ツールバー、ツウィーク、数式・変数ウィンドウを一体化しています。

### 実装済み

- SVGプロットエリア、方眼、拡大縮小、パン
- 選択・移動・複製・削除、元に戻す・やり直す
- フリーハンド、線分、矢印、四角形、円・だ円、文字、数式、多角形
- アレー図（行・列を数値または変数式で指定）
- 線分図・数直線（目盛数、始値・終値を数値または変数式で指定）
- 変数フェーダーとアレー図・線分図のリアルタイム連動
- 数式一覧、簡易入力、詳細入力オーバーレイ、キャンバス配置
- プロジェクトJSONの保存・読込
- 環境設定JSONの保存・読込、ツール表示・左右配置
- IndexedDB自動保存と前回作業の復旧
- PNGスクリーンショットを新規ウィンドウへ出力
- 全画面の発表モード
- Service Workerによる起動後のランタイムキャッシュ
- 関数グラフの無効アイコンとv3表示

### v1.0.0までに強化する項目

- 数式ウィンドウを、複分数・根号・行列等を視覚的に組み立てる構造エディターへ更新
- 点・交点へのスナップ
- 選択要素のリサイズハンドル、回転、前面・背面操作
- 多角形操作の洗練
- アレー図の個別シンボル追加・削除とグループ化
- 線分図の目盛幅・ラベル個別編集
- スクリーンショットの表示範囲・余白指定
- タッチ・ペン操作の実機調整
- GitHub Pages用デプロイ設定とオフラインキャッシュの更新制御

## 開発環境

- TypeScript
- React
- Vite
- SVG（編集・描画）
- Canvas（PNG生成時のみ）
- IndexedDB / localStorage

## 起動

```bash
npm install
npm run dev
```

ブラウザに表示されるローカルURLをEdgeまたはChromeで開きます。

## ビルド

```bash
npm run build
npm run preview
```

`dist` 内の静的ファイルをGitHub Pagesへ配置できます。`vite.config.ts` は相対パス公開向けに `base: './'` としています。

## 操作

- マウスホイール：拡大・縮小
- パンツール：表示位置を移動
- Delete / Backspace：選択要素を削除
- Ctrl/Cmd + Z：元に戻す
- Ctrl/Cmd + Y または Ctrl/Cmd + Shift + Z：やり直す
- 矢印キー：選択要素を1単位移動
- Shift + 矢印キー：選択要素を10単位移動
- 多角形ツール：頂点を順番にクリックし、Enterで確定、Escで中止

## 保存形式

プロジェクトファイルは `*.graphanta.json`、環境設定は `graphanta-settings.json` です。どちらもバージョン情報を持つJSONで、サーバーには送信されません。
