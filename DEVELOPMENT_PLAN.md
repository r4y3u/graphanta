# Graphanta 開発計画

## v0.1 alpha（今回）

画面構造とデータモデルを確立し、基本作図、アレー図、線分図、変数フェーダー、保存、スクリーンショット、発表モードを一つの動作版に統合する。

## v0.2

- 選択枠の8方向リサイズ
- 線端・図形頂点の直接編集
- 点・交点スナップ
- レイヤー順序
- タッチ操作の誤入力防止
- オブジェクト単位の取り消し履歴の精密化

## v0.3

- 構造化数式エディター
- 複分数、平方根、添字、総和、積分、行列
- 数式のSVG描画とPNG出力の一致
- 数式オブジェクトと変数値の表示連動

## v0.4

- アレー図の個別シンボル操作
- 囲み、分割、色分け、部分選択
- 線分図のラベル個別編集、区間ごとの値、括弧・補助線
- 数直線プリセット

## v0.5

- 複数ページまたは複数シーン
- プレゼン用の段階表示
- スクリーンショット範囲指定
- Chromebook、Windows 11、タッチ端末の実機調整

## v1.0.0

- 初期仕様の安定化
- 互換性付き保存形式
- GitHub Pages公開設定
- オフライン更新通知
- キーボード操作・アクセシビリティ確認
- 教育現場向けの安全性・プライバシー説明

## v2までに検討

長さ固定、角度固定、平行・垂直制約、寸法表示、高度な作図補正、発表用の段階表示。

## v3までに実装

関数グラフ、`y=f(x)`、直線、定義域、離散点、変数フェーダーとの曲線連動。
